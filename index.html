🚀 导弹追踪飞机

项目地址：github.com/Aqua65535/Pursuit_Curve

📋 问题背景

一架飞机在二维平面内以恒定速度水平向右飞行。地面发射一枚导弹，导弹始终朝向飞机的当前位置飞行，且导弹速度大于飞机速度。这是一个经典的纯追踪问题，广泛用于军事、机器人与控制理论。

数学描述

  • 飞机速度：vp (水平向右)
  • 飞机初始位置：(xp0, yp0)
  • 导弹速度：vm (满足 vm > vp)
  • 导弹初始位置：(xm0, ym0)，通常取 (0,0)
  • 时间步长：Δt
飞机运动方程：
xp(t+Δt) = xp(t) + vp · Δt
yp(t) = 常数（水平飞行）

导弹采用纯追踪法：速度方向始终指向飞机当前位置。导弹与飞机距离：

d = √((xp - xm)² + (yp - ym)²)

导弹速度分量：

vmx = vm · (xp - xm) / d
vmy = vm · (yp - ym) / d

导弹位置更新：

xm(t+Δt) = xm(t) + vmx · Δt
ym(t+Δt) = ym(t) + vmy · Δt

判定条件 当导弹x坐标 ≥ 飞机x坐标时，认为成功击中。

🧠 解法概述

本项目采用数值积分法（欧拉法）离散化求解微分方程。

  1. 初始化：设置飞机/导弹位置、速度、时间步长。
  2. 循环迭代：计算距离 → 判断击中 → 更新飞机位置 → 计算导弹方向 → 更新导弹位置。
def update_missile(x_m, y_m, x_p, y_p, v_m, dt): dx = x_p - x_m dy = y_p - y_m distance = np.sqrt(dx**2 + dy**2) if distance > 0: vx = v_m * (dx / distance) vy = v_m * (dy / distance) x_m += vx * dt y_m += vy * dt return x_m, y_m

📊 结果分析

1. 理论保证

因为 vm > vp 且导弹始终指向飞机，理论上导弹一定能击中目标。

2. 观察现象

  • 导弹轨迹是一条平滑曲线，从发射点逐渐逼近飞机航线。
  • 随着接近目标，导弹轨迹曲率增大。

3. 参数影响

参数变化影响
导弹速度 ↑击中时间 ↓，轨迹更"直"
飞机速度 ↑击中时间 ↑，追逐距离更长
初始Y坐标差 ↑轨迹弯曲程度 ↑
